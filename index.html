Información sobre mis estudios

Carrera: Ingeniería en Computación

Actualmente estoy cursando el quinto semestre de la carrera de Ingeniería en Computación en la Facultad de Estudios Superiores Aragón (FES Aragón) de la Universidad Nacional Autónoma de México (UNAM). Esta carrera tiene como objetivo formar profesionales capaces de diseñar, implementar y gestionar sistemas de computación, tanto en software como en hardware, y de aplicar tecnologías emergentes para resolver problemas en diversas áreas.

Asignaturas Destacadas

  • Estructuras de Datos: Aprendí a implementar y utilizar estructuras de datos avanzadas, como listas enlazadas, árboles y grafos, para resolver problemas computacionales de manera eficiente.
  • Algoritmos: En esta asignatura, me enfoqué en el diseño y análisis de algoritmos, optimizando su rendimiento y aplicándolos a problemas complejos.
  • Base de Datos: Estudié el diseño, implementación y administración de bases de datos relacionales, incluyendo la creación de esquemas, consultas SQL y la optimización del rendimiento de las bases de datos.
  • Programación Orientada a Objetos (POO): Me enfoqué en el diseño y desarrollo de software utilizando el paradigma de programación orientada a objetos, aprendiendo conceptos como herencia, polimorfismo y encapsulamiento.

Proyectos Realizados

  • Diseño de una Página Web: Participé en el diseño y desarrollo de una página web para un cliente ficticio, utilizando HTML, CSS y JavaScript para crear una interfaz de usuario atractiva y funcional.
  • Implementación de un Sistema de Gestión de Base de Datos: Desarrollé un sistema de gestión de base de datos para una empresa ficticia, utilizando MySQL y realizando consultas complejas para extraer y manipular datos.
  • Proyecto de Programación Orientada a Objetos: Implementé un sistema de gestión de una panaderia utilizando principios de programación orientada a objetos, incluyendo clases, herencia y polimorfismo.

Habilidades Adquiridas

Hasta el quinto semestre, he adquirido las siguientes habilidades:

  • Programación en diversos lenguajes, incluyendo Python, Java y C++.
  • Diseño y análisis de algoritmos eficientes para la resolución de problemas.
  • Desarrollo de páginas web utilizando HTML, CSS y JavaScript.
  • Diseño, implementación y administración de bases de datos relacionales.
  • Desarrollo de software utilizando el paradigma de programación orientada a objetos.
  • Trabajo en equipo y gestión de proyectos en entornos colaborativos.

Habilidades Tecnológicas

Lenguajes de Programación

  • Python: Básico. Conocimientos en la escritura de scripts simples y automatización de tareas.
  • Java: Intermedio. Experiencia en desarrollo de aplicaciones de escritorio y móviles, así como en el uso de la plataforma Java EE para aplicaciones empresariales.
  • C++: Intermedio. Conocimientos sólidos en programación orientada a objetos y desarrollo de aplicaciones con alto rendimiento.
  • JavaScript: Intermedio. Desarrollo de aplicaciones web interactivas utilizando tecnologías como HTML5, CSS3 y frameworks como React.js y Node.js.

Herramientas de Desarrollo

  • Git: Intermedio. Gestión de versiones y colaboración en proyectos mediante Git y plataformas como GitHub.
  • Visual Studio Code: Intermedio. Editor de código para desarrollo en múltiples lenguajes con extensiones para mejorar la productividad.
  • MySQL: Intermedio. Diseño, implementación y administración de bases de datos relacionales, incluyendo la escritura de consultas SQL complejas.
  • Docker: Básico. Conocimiento en la creación y gestión de contenedores para el desarrollo y despliegue de aplicaciones.
  • VirtualBox: Intermedio. Uso de VirtualBox para crear y gestionar máquinas virtuales, facilitando el entorno de desarrollo y pruebas.

Sistemas Operativos

  • Linux: Intermedio. Experiencia en la administración y uso de sistemas operativos basados en Linux, incluyendo la línea de comandos y la configuración de entornos de desarrollo.

Conceptos y Tecnologías Web

  • HTML5: Avanzado. Estructuración de contenido web y desarrollo de interfaces con estándares modernos.
  • CSS3: Básico. Diseño de estilos web con conocimientos fundamentales en CSS3 para la creación de estilos y layout básicos.
  • Frameworks de JavaScript: Básico. Conocimiento en el uso de frameworks como React.js y Node.js para el desarrollo de aplicaciones web.

Redes y Herramientas de Análisis

  • Cisco Packet Tracer: Básico. Simulación de redes para el diseño y análisis de configuraciones de red.
  • Wireshark: Básico. Análisis de tráfico de red para la resolución de problemas y diagnóstico de redes.

Otras Habilidades

  • Programación Orientada a Objetos (POO): Avanzado. Aplicación de principios como herencia, polimorfismo y encapsulamiento en el desarrollo de software.
  • Metodologías Ágiles: Intermedio. Experiencia en la aplicación de metodologías ágiles como Scrum para la gestión de proyectos.
  • Seguridad en Aplicaciones: Básico. Conocimientos en prácticas de seguridad para el desarrollo de aplicaciones seguras.
